# Himanshu Sharma — Portfolio

A clean, single-file developer portfolio that pulls live data from the GitHub API.

Built with React (via Babel standalone) — no build step required. Just edit `index.html` and push to GitHub Pages.

## Features
- Dark / Light theme toggle
- Live GitHub stats and repositories
- Search, sort, and language filter
- Hand-picked pinned projects
- Smooth animations and subtle effects
- Fully responsive

## Quick Edit
1. Open `index.html` directly in GitHub (mobile or desktop)
2. Click the pencil icon ✏️ to edit
3. Update anything you want (pinned repos are defined at the top)
4. Commit → your site updates instantly

## Customization
- Change `GH_USER`, `DISPLAY`, `HANDLE` at the top of the `<script>` section.
- Edit the `PINNED_REPOS` array to feature different projects.
- Modify the About text or add new sections easily.
